Place tes sons ici (formats acceptés : .wav / .ogg / .mp3) :
- hit.*        (collision joueur/obstacle)
- powerup.*    (ramassage de power-up)
- zap.*        (power-up ZAP)
- music.*      (musique de fond, optionnel)

Le code charge le premier fichier trouvé avec ces noms, peu importe l’extension. Garde des fichiers légers.
